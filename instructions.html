Reaction Time Tester

Game Instructions

Game Goal

The goal of the game is to react as quickly as possible when the signal appears. The game measures your reaction time over multiple rounds and calculates your results.

How to Play

  1. Enter your player name on the launcher page.
  2. Select the difficulty level.
  3. Select how many rounds you want to play.
  4. Choose the signal type.
  5. Select optional features such as sound and false-start penalties.
  6. Click Open Game Window.
  7. In the game window, click Start Test.
  8. Wait for the signal area to change.
  9. React as quickly as possible by clicking or pressing the spacebar, depending on the round rule.
  10. Complete all rounds to see your average and best reaction time.

Difficulty Levels

  • Easy: longer and more predictable waiting times
  • Medium: medium waiting times
  • Hard: shorter and less predictable waiting times

Buttons in the Game Window

  • Start Test starts a new reaction test
  • Next Round moves to the next round after a result
  • Save Session stores the current game state
  • Load Session restores a saved session
  • Reset Test clears the current test
  • Back to Settings closes the game window and returns to the launcher

Scoring and Results

  • Faster reactions earn a better score
  • False starts may reduce your score if penalties are enabled
  • The game records each round time
  • At the end, the game shows your best, worst, and average reaction time
