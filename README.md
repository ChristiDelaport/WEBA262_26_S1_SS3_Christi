# WEBA262_26_S1_SS3_Christi
Christi's Reaction Time Game
A simple and fun reaction time game built to test how fast you can respond when the screen changes color.

##  Features

- Random reaction timer
- Instant reaction score in milliseconds
- Simple and clean UI
- Restart and try again 

## Technologies Used

- HTML
- CSS
- JavaScript

##  Preview

Click the screen when the color changes to green as fast as possible.

##  How to Play

1. Press the **Start Game** button.
2. Wait for the game screen to change.
3. Click immediately when it changes.
4. Your reaction time will be displayed.
